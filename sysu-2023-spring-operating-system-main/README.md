# 项目结构

+ `appendix`用于存放一些先修知识的介绍。
+ `labX`用于存放第`X`次实验，在`labX`文件夹下
  + `gallery`用于存放指导材料中的图片资源。
  + `src`用于存放示例代码。
  + `README.md`是指导材料。
+ `env`用于存放特定版本的软件。
+ `references`用于存放参考资料。
